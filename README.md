## Event Management System
![Static Badge](https://img.shields.io/badge/react-blue) ![Static Badge](https://img.shields.io/badge/springboot-green) ![Static Badge](https://img.shields.io/badge/mysql-yellow)

 </br>

This project is a comprehensive event management website built using the React.js, React Bootstrap, Sprngboot,  MySQL and CSS. The application includes multiple features including add event, delete event, update event, search event by ID and view event.

<br/>Features
<br/>Home Page: Include list of added events.
<br/>Event: Details about the added events, user can update,delete and view event details through this page.
<br/>Add Event Page: Include a form to add events for the users. 
<br/>Navigation bar: Consist with logo, search bar and the navigation tabs.
<br/>Search bar: Users can search events by event ID.



## Installation


1. Clone the repo
   ```sh
   git clone  https://github.com/sudeepa99/Event-Management-System
   ```
2. Installing packages
   ```sh
   cd frontend
   cd event_management_system
   npm install

   backend - download and install JDK.
   ```
3. Start the front end server
   ```sh
   cd frontend
   cd event_management_system
   npm start
   ```
4. Start the back end server
   ```sh
   cd event_management
   Import the project to prefered IDE(Intellij IDEA)
   Maven build with relevant dependencies.
   Run the application.
   ```
